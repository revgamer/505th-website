CONTROL REFERENCE

REQUIRED CONTROL CHANGES FOR 505TH OPERATIONS // ARMA 3 + ACE3 + OPTRE

This page lists the keybind changes required before joining 505th operations. Apply these in your Arma 3 options under the listed path. More keybinds will be added as documentation is completed.
Note: "Unbind" means remove the existing bind entirely to prevent conflicts.
🚧  MORE KEYBINDS BEING DOCUMENTED — PAGE WILL BE UPDATED
ACTION KEYBIND EXPLANATION WHERE TO FIND
⚙ REQUIRED CHANGES
CHANGE Throw = GG Double-tap G to throw grenade — prevents accidental mass casualty events Controls → Weapons
BIND Vehicle Quick Mount Easy and fast mounting of vehicles Controls → Addons → ACE Movement
BIND Webknights Melee = Your preferred key Set to any key of your choice — avoid F (fire mode) and G (grenade) Controls → Addons → Webknights Melee
UNBIND Lago Melee Remove to prevent conflict with Webknights Melee Controls → Addons → Lago Melee
⬆ MOVEMENT
DEFAULT WASD Move forward, left, back, right Controls → Movement
DEFAULT Shift+W Sprint — hold Shift while moving forward Controls → Movement
DEFAULT CC Double-tap C — toggle between walk and jog speed Controls → Movement
DEFAULT Z Go prone — reduces profile, increases accuracy Controls → Movement
DEFAULT X Crouch Controls → Movement
🎯 COMBAT & WEAPONS
DEFAULT LMB Fire weapon Controls → Weapons
DEFAULT RMB Aim down sights / use optic Controls → Weapons
DEFAULT CtrlCtrl Double-tap Ctrl — lower / raise weapon Controls → Weapons
DEFAULT R Reload Controls → Weapons
DEFAULT GG Throw grenade — double-tap to prevent accidents Controls → Weapons
DEFAULT 8 Cycle grenade / throw mode selection Controls → Weapons
🔫 WEAPON SLOTS
DEFAULT 1 Primary weapon (rifle / LMG) Controls → Weapons
DEFAULT 2 Secondary weapon (pistol / sidearm) Controls → Weapons
DEFAULT 3 Launcher (rocket / AT weapon) Controls → Weapons
DEFAULT 0 Holster — put weapon away to belt/sling Controls → Weapons
